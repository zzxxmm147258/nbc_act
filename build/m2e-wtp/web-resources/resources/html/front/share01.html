活动策划

课程化：突出教育价值和教育意义，实践教育与学生所学所需结合，注重结果性收获和感受。

体系化：实践活动以项目为单位，根据青少年成长发展特点及学校文化，融合学科，同时从知识、技能、价值三维度开展实践活动。

学生主体：结合学生认知发展为基础，实践以学生为主体，高参与度、融合趣味性。
